IndexGIXS is software to index grazing-incidence x-ray scattering patterns of molecular and nanostructured thin films, as obtained on x-ray area detectors such as CCDs, pixel array detectors and others. Many generic detector formats based on TIFF are supported, as well as beamline specific formats. 

For further information and a program description please consult the following open access papers:

"Probing Functional Thin Films with Grazing Incidence X-Ray Scattering: The Power of Indexing", Crystals 2025, 15, 63.   
https://doi.org/10.3390/cryst15010063

"indexGIXS – software for visualizing and interactive indexing of grazing-incidence scattering data", ChemRxiv 6/2025, Version 4.
https://doi.org/10.26434/chemrxiv-2021-j1bww-v4

Detailed strategies to find an indexation for the surface unit cell of low-symmetry molecular thin films are discussed in
"Indexing 2D Powders and Lagrange-Gauss Reduction", Crystals 2026, 16, 43    https://doi.org/10.3390/cryst16010043

If you use indexGIXS for your research, please cite these papers.  

Installation: IndexGIXS is based on the free Scilab software which is open source and can be downloaded at www.scilab.org. Scilab also provides an interactive development environment and extensive documentation. IndexGIXS version 3A features a major clean-up and improvement to the code. 

Compatibility: Version 3A is compatible with the current Scilab distribution scilab-2026.0.1 on Windows 10 and Windows 11 Home editions. There is an issue with the new Scilab graphics engine in the Windows 11 Enterprise edition - I recommend rolling back to the earlier Scilab version scilab-2023.1.0 which is still available at www.scilab.org as legacy code and compatible with Windows 11 Enterprise, albeit with slower graphics.
