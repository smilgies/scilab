IndexGIXS is software to index grazing-incidence scattering patterns, as obtained on x-ray area detectors such as CCDs, 
pixel array detectors and others. Many generic detector formats are supported, as well as beamline specific formats. 
For further information please consult the papers below which can also be found in the literature folder.

"Probing Functional Thin Films with Grazing Incidence X-Ray Scattering: The Power of Indexing"
Crystals 2025, 15, 63.   https://doi.org/10.3390/cryst15010063

"indexGIXS – software for visualizing and interactive indexing of grazing-incidence scattering data", ChemRxiv 6/2025, Version 4.   DOI: 10.26434/chemrxiv-2021-j1bww

Some detailed strategies to find an indexation for the surface unit cell of 2D powders are discussed in
"Indexing 2D Lattices and Lagrange-Gauss Reduction", Crystals 2026, 16, 43    https://doi.org/10.3390/cryst16010043

If you use indexGIXS for your research, please cite these papers.  

IndexGIXS is based on the free Scilab software which is open source and can be downloaded at Scilab.org. IndexGIXS version 3A 
features a major clean-up and improvement to the code. The version is compatible with the current Scilab distribution 
scilab-2026.0.1 on Windows 10 and Windows 11 Home editions. There is an issue with the new Scilab graphics engine in the 
Windows 11 Enterprise edition - I recommend rolling back to the earlier Scilab version scilab-2023.1.0 which is still 
available at scilab.org as legacy code and compatible with Windows 11 Enterprise, albeit with slower graphics.
